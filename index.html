Journey towards Coding

My journey for coding started as a result of having a need to prove to myself that I was not a faliure. This was due to the fact the last thing my Headteacher telling me that I should just go into work instead of trying to get into University.
The sheer idea that a person who's main job was to help students with their education, had told me that I should stop trying ignited my desire to prove everyone wrong and after completing a foundation year in Computing achieving highest in the course.
I decided to pursue Msci Computer Science as I felt as if for the first time I found something that I not only did well academically but also enjoyed.

Finite State Automata

For my first ever coding project I decided to make a Finite-state machine. Which I coded with Python which helped me to develop an understanding for how computers will behave when simulating sequential logic.

Video Game

For my term long coursework I was assigned to create a video game using the Universities game engine with java. During this time I developed a good understanding on object orientated programming.
Due to having to learn how, I could accomodate my code for things such as collision between objects but also how to manipulate the engine in a way to achieve my desired results of having a challenging but fun platformer.
